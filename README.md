# adarsha.dev

My personal site — portfolio, blog, and a chat widget that can answer questions about whatever page you're on.

Built with Next.js, Tailwind, and MDX (contentlayer2). Deployed on Vercel. The chat runs on AI SDK.

## Run it locally

Node 22+, pnpm.

```bash
git clone https://github.com/adarshaacharya/adarsha.dev
cd adarsha.dev
pnpm install
cp .env.example .env.local   
pnpm dev
```
